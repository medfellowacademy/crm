## 🎨 Color System

### Lead Segments
- 🔴 **Red** = Hot leads (urgent)
- 🟠 **Orange** = Warm leads (follow up soon)
- 🟢 **Green** = Cold/Enrolled leads

### Follow-up Dates
- 🔴 **Red** = Overdue (action needed!)
- 🟠 **Orange** = Today (act now)
- ⚪ **Gray** = Future dates

### Revenue Display
- 🟢 **Green** = Actual revenue (enrolled)
- 🟠 **Orange** = Expected revenue (potential)

## ⚡ Quick Actions

### Bulk Operations
1. Select leads with checkboxes ☑️
2. Click "Bulk Update" button
3. Choose what to update
4. Click "Update All"

**Bulk Update Options**:
- Status (Follow Up, Warm, Hot, etc.)
- Assigned To (counselor)
- Follow-up Date

### Quick Filters
Click segment buttons to filter:
- **All** - Show everything
- **Hot 🔥** - Only urgent leads
- **Warm ⚡** - Medium priority
- **Today** - Created today
- **Overdue** - Needs attention NOW

### Export Data
1. Apply filters (optional)
2. Click "Export" button
3. CSV downloads automatically
4. Open in Excel/Sheets

## 🆘 Troubleshooting

### Data Not Loading?
1. Check backend is running
2. Click Refresh button
3. Check browser console
4. Verify API endpoint

### Bulk Update Not Working?
1. Ensure leads are selected
2. Fill at least one field in form
3. Check backend logs
4. Verify permissions

### Export Not Downloading?
1. Disable popup blockers
2. Check download folder
3. Try different browser
4. Check file permissions
