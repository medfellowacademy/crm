## 🆘 Support & Help

### Common Issues

**Q: Page not loading?**
```bash
# Check if frontend is running
lsof -ti:3000

# Restart if needed
cd crm/frontend
npm start
```

**Q: Data not showing?**
```bash
# Check backend
curl http://localhost:8000/health

# Restart if needed
cd crm/backend
source venv/bin/activate
uvicorn main:app --host 0.0.0.0 --port 8000
```

**Q: Bulk update not working?**
- Ensure leads are selected (checkboxes)
- Fill at least one field in form
- Check browser console for errors
